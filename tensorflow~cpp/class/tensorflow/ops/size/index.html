tensorflow::ops::Size

#include <array_ops.h>

Returns the size of a tensor.

Summary

This operation returns an integer representing the number of elements in input.

For example:

```prettyprint 't' is [[[1, 1,, 1], [2, 2, 2]], [[3, 3, 3], [4, 4, 4]]]]

size(t) ==> 12 ```

Arguments:

  • scope: A Scope object

Returns:

  • Output: The output tensor.
