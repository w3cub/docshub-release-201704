tensorflow::ops::Polygamma

#include <math_ops.h>

Compute the polygamma function \(^{(n)}(x)\).

Summary

The polygamma function is defined as:

``` ^{(n)}(x) = {d^n}{dx^n} (x) ``` where \((x)\) is the digamma function.

Arguments:

  • scope: A Scope object

Returns:

  • Output: The z tensor.
