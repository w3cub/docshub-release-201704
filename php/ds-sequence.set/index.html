Ds\Sequence::set

(PECL ds >= 1.0.0)

Ds\Sequence::set — Updates a value at a given index.

Description

abstract public void Ds\Sequence::set ( int $index , mixed $value )

Updates a value at a given index.

Parameters

index

The index of the value to update.

value

The new value.

Return Values

No value is returned.

Errors/Exceptions

OutOfRangeException if the index is not valid.

Examples

Example #1 Ds\Sequence::set() example

<?php
$sequence = new \Ds\Vector(["a", "b", "c"]);

$sequence->set(1, "_");
print_r($sequence);
?>

The above example will output something similar to:

Ds\Vector Object
(
    [0] => a
    [1] => _
    [2] => c
)

Example #2 Ds\Sequence::set() example using array syntax

<?php
$sequence = new \Ds\Vector(["a", "b", "c"]);

$sequence[1] = "_";
print_r($sequence);
?>

The above example will output something similar to:

Ds\Vector Object
(
    [0] => a
    [1] => _
    [2] => c
)
