end

This attribute defines an end value for the animation that can constrain the active duration.

The attribute value is a semicolon separated list of values. Each individual value can be of the same type as the ones defined for the begin attribute.

Usage context

Categories Animation timing attribute
Value <End-value-list>
Animatable No
Normative document SVG 1.1 (2nd Edition)

For the types of value allowed in the <end-value-list> see the begin attribute.
